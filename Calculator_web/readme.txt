# Basic Calculator

This is a basic calculator application built with HTML, CSS, and JavaScript. It provides standard calculator functionality for addition, subtraction, multiplication, and division.

## Usage

1. Open the `index.html` file in your preferred web browser.
2. The calculator interface will be displayed.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

## How to Use

- Press the numeric buttons (0-9) to input numbers.
- Press the operator buttons (+, -, *, /) to perform calculations.
- Press the decimal button (.) to add a decimal point.
- Press the "C" button to clear the display.
- Press the "=" button to calculate the result.

## Examples

- To add: `2 + 3`, press `2`, `+`, `3`, and then `=`.
- To subtract: `5 - 2`, press `5`, `-`, `2`, and then `=`.
- To multiply: `4 * 3`, press `4`, `*`, `3`, and then `=`.
- To divide: `6 / 2`, press `6`, `/`, `2`, and then `=`.

## Customization

You can customize the calculator's appearance and colors by modifying the CSS file (`cal_css.css`).

## Dependencies

- [Digital-7 Font](fonts/digital-7.css) - Used for the display font.
